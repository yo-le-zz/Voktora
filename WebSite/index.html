Fonctionnalités

Tout ce dont vous avez besoin

Voktora centralise la gestion de vos projets dans une interface unifiée et extensible.

🗂
Vue liste & grille
Basculez entre une liste compacte et des cartes visuelles avec icône personnalisable, badge langage et statut.
🔐
Vault AES-256
Stockage chiffré de tokens GitHub, clés SSH et API keys. Master password dérivé PBKDF2, 480 000 itérations.
⚡
Profils d'exécution
Définissez commande, env vars, dossier de travail et scripts pre/post run par projet. Lancement en un clic.
🪝
Hooks
Automatisations shell ou Python sur 7 événements : on_create, on_open, on_git_push, on_clone, etc.
📸
Snapshots
Capturez l'état complet d'un projet, restaurez-le en un clic, comparez deux snapshots ligne par ligne.
🧱
Templates
Python, C++, Web App, Discord Bot, Minecraft Mod, ou vide. Structure + git init + premier commit automatiques.
📊
Dashboard santé
Score 0–100 par projet. Détecte .gitignore manquant, commits en retard, branches non mergées, inactivité.
🧩
Plugins extensibles
Ajoutez un .py dans data/plugins/ pour enregistrer commandes, boutons et hooks. Rechargement à chaud.
🐙
GitHub App & OAuth
Authentification via GitHub App (JWT RS256, token auto-renouvelé) ou OAuth App Device Flow classique.
🧠
Smart commit
Génère des messages Conventional Commits sans IA en analysant les fichiers modifiés. feat, fix, docs, style…
🌙
Thème Catppuccin
Interface sombre Catppuccin Mocha. PySide6, polices cross-platform, transitions fluides.
📦
Packages natifs
.deb pour Debian/Ubuntu, .msi pour Windows. Installation native sans Python requis sur la machine cible.

Installation

Démarrez en 30 secondes

Packages natifs pour Linux et Windows. Aucun Python requis sur la machine cible.

🐧 Linux (Debian / Ubuntu / Mint)

# Télécharger et installer wget https://github.com/yo-le-zz/Voktora /releases/latest/download /voktora_1.0.0_amd64.deb sudo dpkg -i voktora_1.0.0_amd64.deb # Si dépendances manquantes sudo apt-get install -f # Lancer voktora

🪟 Windows 10 / 11

# 1. Télécharger Voktora_1.0.0_x64.msi # depuis GitHub Releases # 2. Double-cliquer sur le .msi # → Next → Install → Finish # 3. Voktora est dans # C:\Program Files\Voktora\ # et dans le menu Démarrer # Désinstaller : # Paramètres → Applications → Voktora

🐍 Depuis les sources (Python)

git clone https://github.com/yo-le-zz/Voktora cd Voktora pip install uv uv pip install pyside6 cryptography python src/main.py

🔨 Compiler soi-même

# Linux (.deb) bash packaging/build_deb.sh 1.0.0 # Windows (.msi) python packaging/build_msi.py 1.0.0 # Voir docs/BUILD.md pour les # prérequis Nuitka + WiX
